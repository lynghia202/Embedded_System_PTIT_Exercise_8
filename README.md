# Embedded_System_PTIT_Exercise_8
## Giới thiệu
Bài tập 8 - ADC Configuration
## Yêu cầu:
- Cấu hình một kênh ADC để đọc giá trị từ một biến trở hoặc cảm biến nhiệt độ.
- Chuyển đổi giá trị này thành điện áp (mV) và hiển thị lên terminal."
## Phần cứng sử dụng
- STM32F103C8T6
- USB to TTL
- Cảm biến quang trở
